# 车辆管理系统 - 部署说明

## 项目结构

```
vehicle_api/
├── main.py              # 后端主程序
├── index.html           # 前端页面
├── vehicle_management.db # SQLite数据库（自动创建）
├── requirements.txt     # Python依赖
├── start.bat            # Windows启动脚本
├── start.sh             # Linux/Mac启动脚本
└── 部署说明.md          # 本文件
```

## 一、环境要求

- Python 3.8+
- 现代浏览器（Chrome/Firefox/Edge）

## 二、安装依赖

```bash
pip install fastapi uvicorn sqlalchemy pydantic openpyxl
```

或

```bash
pip install -r requirements.txt
```

## 三、启动方式

### 方式1：命令行启动

```bash
python main.py
```

### 方式2：使用启动脚本

- Windows: 双击 `start.bat`
- Linux/Mac: `./start.sh`

## 四、访问系统

启动后，打开浏览器访问：

- 前端页面：http://localhost:8000/index.html
- API文档：http://localhost:8000/docs

## 五、打包部署

### 打包为ZIP文件

将以下文件打包：
- main.py
- index.html
- requirements.txt
- start.bat / start.sh
- 部署说明.md

### 部署到新机器

1. 解压ZIP文件
2. 安装依赖：`pip install -r requirements.txt`
3. 运行：`python main.py`
4. 访问：http://localhost:8000

## 六、数据备份

数据库文件：`vehicle_management.db`

点击界面"备份"按钮可自动备份，备份文件保存在 `backups/` 目录。

## 八、默认端口

- 后端服务：8000
- 如需修改，编辑 main.py 最后一行

## 九、常见问题

1. **端口被占用**：修改 main.py 中的 port=8000 为其他端口
2. **跨域问题**：已自动处理，无需配置
3. **中文乱码**：确保文件编码为 UTF-8
