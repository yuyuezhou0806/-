# PDF 盖章工具

## 功能

1. **竖向骑缝章**：所有页右边缘同一 x 位置盖印章切片，叠齐后是一个完整圆形章（真实骑缝章效果）
2. **乙方公章自动放大**：检测 PDF 中已有的小公章（约 60pt）并放大到 130pt 重盖
3. **清理旧章**：删除老版错误的横切骑缝章细条（90×9pt）

## 启动方式

- **GUI（推荐）**：双击 `run.bat` 或 `PDF盖章工具.exe`
- **命令行**：`python pdf_stamping.py`

## 使用流程

1. 选择 PDF 文件或文件夹（批量）
2. 选择透明背景印章 PNG
3. 选择输出目录
4. 勾选要执行的处理选项（默认全开）
5. 点击「开始处理」

## 默认参数

- 骑缝章高度：页面高度的 **20%**
- 切片可见度：**80%**（20% 出血到页外）
- 乙方章宽度：**130pt** (~4.6cm)
