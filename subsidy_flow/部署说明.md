# 广州市中小企业数字化转型补贴申报系统 - 部署说明

## 一、方案概述

使用 **内网穿透（cpolar）** 方案，零成本、快速上线，适合给客户远程展示。

**原理**：
1. 你的 Windows 电脑运行补贴申报系统（本地服务）
2. cpolar 把本地 8000 端口映射到公网，生成一个外部可访问的链接
3. 把链接发给客户，客户在任何地方都能打开

---

## 二、环境要求

- Windows 10/11 电脑一台（需要长期开机）
- 已安装 Python 虚拟环境（.venv）
- 网络通畅（能访问外网）

---

## 三、首次配置（只需做一次）

### 1. 注册 cpolar 账号

打开 https://www.cpolar.com/ ，用手机号注册一个免费账号。

注册后登录控制台：https://dashboard.cpolar.com/ ，在左侧菜单找到：
**验证 → 您的 authtoken**

复制那一串 token（类似 `abc123def456ghi789`）。

### 2. 配置 cpolar authtoken

打开命令行（CMD 或 PowerShell），进入 `subsidy_flow` 文件夹：

```cmd
cd C:\Users\admin\cc_test\subsidy_flow
```

运行：

```cmd
cpolar.exe authtoken 你复制的authtoken
```

看到 `Authtoken saved to configuration file` 表示成功。

### 3. 修改默认密码（重要！）

系统默认有两个账号，公网使用前务必修改密码：

| 账号 | 默认密码 | 角色 |
|------|----------|------|
| admin | admin123 | 管理员 |
| agent | agent123 | 申报专员 |

启动系统后登录 `admin`，尽快：
1. 在页面右上角点击管理员名称
2. 新增自己的工作人员账号
3. 把默认账号密码改掉，或不再使用默认账号

## 四、日常启动（每次使用）

### 方法一：双击启动（推荐）

直接双击 `subsidy_flow` 文件夹里的 **`start.bat`**。

会弹出一个黑色命令行窗口，显示：

```
========================================
  广州市中小企业数字化转型补贴申报系统
========================================

[1/2] 正在启动本地服务...
[2/2] 正在启动公网穿透...
```

稍等几秒，cpolar 会输出类似：

```
Forwarding  https://a1b2c3d4.cpolar.io -> http://localhost:8000
```

**`https://a1b2c3d4.cpolar.io`** 就是你的公网链接，复制发给客户即可。

### 方法二：手动分步启动

如果 `start.bat` 有问题，可以手动开两个命令行窗口：

**窗口 1 - 启动系统：**
```cmd
cd C:\Users\admin\cc_test\subsidy_flow
..\.venv\Scripts\python.exe -m uvicorn main:app --host 0.0.0.0 --port 8000
```

**窗口 2 - 启动穿透：**
```cmd
cd C:\Users\admin\cc_test\subsidy_flow
cpolar.exe http 8000
```

## 七、常见问题

### Q1：电脑关机后链接还能用吗？
不能。电脑关机或服务关闭后，公网链接就失效了。需要那台电脑**一直开机**。

### Q2：多台电脑能同时管理吗？
可以。只要那台运行系统的电脑开机，你们办公室其他电脑用内网 IP（如 `http://192.168.1.100:8000`）就能访问；外面的客户用 cpolar 公网链接访问。

### Q3：数据存在哪里？
所有数据存在 `subsidy_flow/subsidy_flow.db` 这个 SQLite 数据库文件里。**务必备份这个文件**，可以定期复制到 U 盘或网盘。

### Q4：免费版域名每次变，抖音链接怎么办？
有两个办法：
1. 抖音个人简介或视频评论区，每次启动后更新链接
2. 花 9元/月买 cpolar 付费版，绑定固定域名，一劳永逸

## 八、技术支持

如遇问题：
1. 检查 `.venv` 虚拟环境是否存在
2. 检查 8000 端口是否被占用（`netstat -ano | findstr 8000`）
3. 检查 cpolar authtoken 是否配置成功（`cpolar.exe authtoken`）
