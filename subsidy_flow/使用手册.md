# 广州市中小企业数字化转型补贴申报系统 - 使用手册

## 一、启动系统

### 方式1：双击启动（最简单）
进入 `subsidy_flow` 文件夹，**双击 `start.bat`**。

黑窗口会显示：
```
========================================
  广州市中小企业数字化转型补贴申报系统
========================================
[1/2] 正在启动本地服务...
[2/2] 正在启动公网穿透...
Forwarding  https://a1b2c3d4.cpolar.io -> http://localhost:8000
```

**复制 `https://a1b2c3d4.cpolar.io` 这个链接发给客户。**

自己使用的话，打开浏览器访问：
- **本机**：`http://localhost:8000`
- **局域网其他电脑**：`http://你的内网IP:8000`（如 `http://192.168.1.5:8000`）

### 方式2：命令行启动
```bash
cd subsidy_flow
../.venv/Scripts/python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

## 二、你自己怎么用（后台管理）

### 登录
打开页面后，点击右上角 **"工作人员登录"**，输入：
- 账号：`admin`
- 密码：`admin123`

登录后左上角菜单变为：**工作台 | 客户 | 项目 | 获客页面**

### 日常工作流程

```
Step 1: 录入客户
    ↓ 点击「客户」→「+ 新增客户」→ 填写企业信息

Step 2: 创建申报项目
    ↓ 进入客户详情 →「+ 为该客户新建项目」
    ↓ 或点击「项目」→「+ 新建项目」

Step 3: 添加投资明细
    ↓ 项目详情页 →「管理投资明细」
    ↓ 逐项录入：软件、云服务、设备等（系统会自动标出可补贴/不可补贴）
    ↓ 右侧实时显示预估补贴金额

Step 4: 提交合规校验
    ↓ 确认合同类型和签订日期无误后
    ↓ 点击「已签合同」按钮，系统自动校验是否符合政策要求

Step 5: 跟踪进度
    ↓ 后续根据实际进展点击状态流转：
    ↓ 实施中 → 待审计 → 审计完成 → 待验收 → 已验收 → 补贴已批 → 补贴已发
```

### 各页面功能说明

| 页面 | 用途 |
|------|------|
| **工作台** | 看数据总览：剩余天数、项目总数、预估补贴总额、各状态项目数 |
| **客户** | 管理所有服务企业，支持搜索 |
| **项目** | 管理所有申报项目，可按状态筛选 |
| **项目详情** | 看合同信息、投资明细、补贴计算结果、操作日志、上传附件 |
| **投资明细** | 增删改费用项，实时计算可补贴金额 |

## 三、给客户用的（公开工具）

客户打开链接后**不需要登录**，可以直接使用：

### 1. 补贴计算器
输入：
- 数字化等级（二级/三级/四级）
- 预计投入金额

输出：
- 预估补贴金额
- 补贴上限提醒
- 计算说明

**场景**：客户在抖音看到视频，点击链接进来，30秒算出自己能拿多少补贴，激发咨询意向。

### 2. 合同合规校验器
输入：
- 所属行业（8大行业之一）
- 合同类型（两方/多方）
- 合同签订日期

输出：
- ✅ 合规 / ❌ 不合规
- 具体原因（如：签订日期早于2025-11-01）
- 预警提醒

**场景**：客户已经签了合同，不确定能不能报补贴，一测便知。

### 3. 政策速览
10条结构化政策要点，折叠展开，重点数字红色高亮。

**场景**：客户想快速了解政策全貌，不用读冗长的政府文件。

## 四、实用技巧

### 技巧1：快速估算补贴
不用创建项目，直接在获客落地页用「补贴计算器」算给客户听，谈成后再录入系统。

### 技巧2：投资明细实时试算
在「投资明细管理」页面，添加费用时会实时显示：
- 可补贴合计（绿色）
- 不可补贴合计（红色）
- 预估补贴（蓝色大字）

这样和客户对账时，可以现场调整各项费用，看补贴怎么变。

### 技巧3：附件管理
每个项目可以上传合同扫描件、发票照片、审计报告等：
- 项目详情页 → 右侧「附件」区域 → 选择文件上传
- 支持 PDF、图片、Word 等格式

### 技巧4：操作日志留痕
系统会自动记录谁、什么时候、做了什么操作（创建、修改、状态变更）。
- 方便团队协作时追责
- 也方便回顾和客户沟通的时间线

### 技巧5：状态颜色识别
项目列表中不同状态有不同颜色标签：
- 🟡 草稿 / 进行中
- 🟢 已验收 / 补贴已批 / 补贴已发
- 🔴 已驳回

一眼就能看出哪些项目卡住了。

## 六、常见问题

**Q：页面打不开？**
- 检查黑窗口是否还在运行
- 检查是不是输错了地址（本地是 `localhost:8000`，局域网要换成内网IP）

**Q：忘记密码？**
- 目前需要手动重置，联系管理员或重新初始化数据库

**Q：能同时开多个项目吗？**
- 可以，一个客户可以有多个申报项目

**Q：补贴金额算错了？**
- 检查费用类别是否选对（培训费、维保费属于「不可补贴」）
- 检查数字化等级是否选对（二级60万/三级200万/四级500万上限）
