## 项目作品(新增模块 — 这是 AI 应用工程师面试的重头)

> *为什么加这一节*:工作经历里写公司项目相对克制(避免给现公司添麻烦),但 AI 应用工程师岗位 HR 和面试官最想看的就是"你独立做过哪些 AI 应用"。把可公开的、跟 AI 强相关的几个挑出来集中展示。

### 1. BPM 内部系统 RPA 自动化套件 ★
针对公司 BPM / CRM 系统,用 Selenium 把月度高频操作做成 Excel 数据驱动的自动化套件,覆盖 18+ 个流程(工作量汇报、商机登记、成员添加、项目延期、PDF 命名等)。其中 `bpm_reporter_fixed.py` 以 Page Object 模式重构,带重试装饰器、日志系统、dataclass 配置,可作为团队 RPA 模板。
**技术栈**:Python 3.10 + Selenium 4 + openpyxl + PyPDF2 + Tesseract OCR
**亮点**:18 脚本 / ~3600 行,每月节省 ~12 小时手工操作,准确率 > 99%。

### 2. IDI 缺陷速查 Web App
工程现场质检场景的移动优先速查工具。从一份 800+ 页 docx 自动提取 599 条结构化数据,前端纯静态(HTML + 原生 JS + Tailwind),支持模糊搜索、规范号检索、命中关键词高亮、一键复制问题+建议。HTTP 内网环境用 fallback 复制方案,iOS Safari 兼容。
**技术栈**:Python (python-docx, openpyxl) + 原生 Web + Tailwind CDN
**亮点**:从需求改判("做平台"→"做手机速查")到形态落地不到一天。

### 3. 合同自动填充与电子盖章工具集
针对建设工程检测合同场景,实现"网页截图 → OCR → 填表 → PDF 盖章"端到端自动化。骑缝章算法基于"所有页同一 x 位置切片"复原真实纸面对齐效果。
**技术栈**:PaddleOCR + python-docx + win32com + PyMuPDF
**亮点**:替代每月 ~50 份合同的手工填写,准确率 > 95%(剩余字段用户简单复核)。

### 4. 补贴申报管理系统(SaaS 形态)
基于规则引擎的小型 B 端业务系统,FastAPI + SQLite + 原生模板渲染,PyInstaller 打包单文件 .exe 一键部署,cpolar 内网穿透。
**技术栈**:FastAPI、SQLAlchemy、JWT、PyInstaller、cpolar

### 5. AI 文生图工作流集成
本地部署 React + Vite + fal.ai 客户端的文生图 playground,配合 ComfyUI + PuLID 工作流处理人脸保留场景。
**技术栈**:React 19 + Vite 6 + Tailwind + @fal-ai/client + ComfyUI + PuLID
**用途**:为公司 / 个人内容创作场景产出素材。
