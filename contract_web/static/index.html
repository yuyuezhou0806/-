    使用说明
    

      
1. 在「上海市建设工程项目信息报送系统」找到项目页面 → 截图保存为图片。

      
2. 在本页上传图片,等待几秒。

      
3. 自动识别建设单位 / 施工单位 / 工程名称 / 报建编号 / 建筑面积等字段并填入 Word 模板。

      
4. 下载 .docx 后打开 Word 检查,补充剩余的字段(总承包、甲方银行账号等)。检测类别可在上传前勾选。

      
本工具仅供内部使用,数据不会发送到外部服务器。
